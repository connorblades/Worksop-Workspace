## Submission workflow

For each directory:

1. **Search first.** Look up "Worksop Workspace" on the directory. If a listing already exists (often auto generated from public data), **claim** rather than create. Claiming inherits any age and avoids duplicates.
2. **Use the canonical NAP** from the top of this file. Copy paste, do not retype. One typo on one directory is enough to suppress ranking.
3. **Paste the canonical short description.** Tweak only if the directory has a character limit smaller than 720.
4. **Upload the primary photo.** Same image across all directories.
5. **Link out to the right page.** Mix it up: roughly half should link to the homepage, half split across `/membership`, `/meeting-room`, `/private-offices`. A directory profile that only ever sends to the homepage is a smaller signal than one that points at the deeper page that matches the directory's purpose.
6. **Note the listing URL once live** so future audits can find it. Copy paste back into this tracker.
7. **Update the status column.** Statuses: Not started, In progress, Submitted, Live, Claimed, Verified, Rejected, Needs payment, Skipped.
8. **Record the login email** used. Future you will thank you when a directory wants verification two years later.

## Two month NAP audit

Run sixty days after the first submission round. Repeat every six months.

1. Google search `"Worksop Workspace"` with the quotes. Cross check the first three pages.
2. For any listing with the wrong address, phone or business name: click in, claim and correct. Record in this tracker.
3. Google search `"30 Carlton Road" Worksop`. Catch orphan listings that name the address but not the business name correctly.
4. Note every fix made in the change log below.
