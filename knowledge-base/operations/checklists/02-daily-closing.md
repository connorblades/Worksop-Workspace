# Daily Closing Checklist

**Worksop Workspace** · Operations · *Phase 1 reference*

Status: Draft v1 — needs Connor review · Last updated: 14 May 2026

---

Tick through this every evening. Full procedure with reasoning in `../sops/sop-002-closing-the-space.md`.

**Start 30 minutes before official closing.**

---

## Members

- [ ] Final floor walk done — everyone wrapping up
- [ ] Member app QR check-in log empty (everyone scanned out)
- [ ] Lost & found items collected to reception

## Reset

- [ ] Hot desks wiped down
- [ ] Chairs realigned
- [ ] Tables back in place
- [ ] Meeting room / podcast room reset if used today

## Refreshments

- [ ] Coffee machine shut down + grounds emptied + drip tray rinsed
- [ ] Milk frother wand wiped
- [ ] Cups / plates washed or dishwasher run
- [ ] Kettle emptied
- [ ] Fridge checked, anything past date discarded
- [ ] Counter wiped down

## Bins and waste

- [ ] All internal bins emptied
- [ ] Waste taken to external bins, separated correctly
- [ ] Bin liners replaced

## Tech and equipment

- [ ] Communal computers / printer shut down
- [ ] Room TVs and equipment powered off
- [ ] Wi-Fi router LEFT ON (never switched off)
- [ ] No laptops or chargers left out unattended

## Environment

- [ ] Heating set to overnight mode
- [ ] All lights off (except safety lighting)
- [ ] All windows closed
- [ ] Internal doors closed as appropriate

## Lock up

- [ ] Final walk of every room — no members, no taps running, no appliances on
- [ ] Alarm set
- [ ] Front door locked
- [ ] Front door tested (definitely locked)

---

If something on this list keeps catching you out or feels missing, edit it. This is a living document.
