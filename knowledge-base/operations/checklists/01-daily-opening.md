# Daily Opening Checklist

**Worksop Workspace** · Operations · *Phase 1 reference*

Status: Draft v1 — needs Connor review · Last updated: 14 May 2026

---

Tick through this every morning. Full procedure with reasoning in `../sops/sop-001-opening-the-space.md`.

**Arrive 30 minutes before official opening.**

---

## Access

- [ ] Unlock front door
- [ ] Disarm alarm
- [ ] Main lights on

## Environment

- [ ] Heating on (or check thermostat in summer)
- [ ] Quick ventilation check, close any open windows before members arrive
- [ ] Visual sweep — desks tidy, chairs aligned, bins empty
- [ ] Toilets clean and stocked

## Technology

- [ ] Wi-Fi router on and broadcasting
- [ ] Test connection from a laptop — full speed
- [ ] Printer powered up (if used today)
- [ ] Room equipment ready (only if a booking is scheduled today)

## Refreshments

- [ ] Coffee machine on and warming up
- [ ] Bean hopper refilled if low
- [ ] Milk stock checked
- [ ] Tea, sugar, water stocked
- [ ] Refreshment area wiped down

## Front-of-house

- [ ] Welcome pack stack at reception
- [ ] Day's bookings checked on member app
- [ ] Door unlocked for members
- [ ] Greeting ready

---

If something on this list keeps catching you out or feels missing, edit it. This is a living document.
